🧠 Ce este AI?

Inteligența artificială (AI) este capacitatea mașinilor de a imita inteligența umană pentru a îndeplini sarcini și de a se îmbunătăți iterativ pe baza informațiilor colectate.

🤖 Învățare automată

Învățarea automată (ML) este o subcategorie a AI care permite sistemelor să învețe și să se îmbunătățească din experiență fără a fi programate explicit.

📊 Exemple simple

Filtrele de spam, recomandările Netflix, asistenții vocali (Siri, Alexa) folosesc AI în fiecare zi.

🔬 Rețele neuronale convoluționale (CNN) 📈 Transformers și atenție 🧮 Algoritmi de optimizare 📚 TensorFlow și PyTorch ⚙️ Machine Learning operaționalizat (MLOps)
